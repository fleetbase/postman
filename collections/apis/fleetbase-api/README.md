# Fleetbase API Collection

Reference collection for the core Fleetbase platform API.

## Variables

- `base_url`
- `namespace`
- `api_key`

## Auth

Requests inherit bearer token authentication from the collection. Set `api_key` locally or through Postman Vault.

## Coverage

This collection is based on the exported Fleetbase API Postman collection and uses `{{base_url}}/{{namespace}}` for request URLs.
